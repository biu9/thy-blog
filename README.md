## 基于react的全栈博客的前端代码

### about
前端部分采用React+Typescript

- 使用react-router进行路由控制，使用localStorage储存token
- 使用remarkable来解析markdown语法
- 用了react hooks来模拟组件的生命周期
- 在登录界面&输入界面用了material UI的一些组件
